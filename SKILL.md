# 记忆管理（内部使用）

## 自动触发
- 说"记住..." → 自动保存
- 问"之前..." → 自动搜索注入
- 对话>5轮 → 自动总结

## 工作方式
此技能**自动运行**，无需手动调用：
1. 检测记忆指令 → 自动保存到 SQLite
2. 检测查询 → 自动语义搜索
3. 自动将相关记忆注入上下文

## 记忆类型
- **用户画像**: ~/.opencode-mem/memory.db (跨项目)
- **项目记忆**: 同上，按项目路径区分
- **会话记忆**: 临时，自动清理
